## 📌 到學校後要做的事（約 10 分鐘，照 backend/後台建置說明.md）

1. 用數辦的 Google 帳號開 [sheets.new](https://sheets.new) 建試算表「數辦填報系統」。
2. 擴充功能 → Apps Script：
   - `Code.gs` 貼上 `backend/Code.gs` 的內容
   - 新增 HTML 檔命名 **Index**，貼上 `backend/Index.html` 的內容
   - （兩個檔案都可以在 GitHub 網頁上開啟後按「Raw」複製，不用回這台筆電）
3. 部署 → 新增部署 → 網頁應用程式：
   **執行身分：我**／**存取權：網域內所有人** → 複製 `/exec` 網址。
4. **權限表不需要 Excel**：在試算表左下新增工作表命名「權限表」，
   選單「檔案 → 匯入 → 上傳」選 `backend/權限表範本.csv`（GitHub 可先下載），
   匯入位置選「插入新工作表」再改名，或直接貼到權限表工作表。
   然後把各校填報人的縣網 Google Email 填入 Email 欄（可先只填幾校測試）。
5. 測試：開 `/exec` 網址，用已填入權限表的帳號登入應可填報；
   未授權帳號應看到「尚未授權」畫面（這是正確行為）。
6. 把 `/exec` 網址交給 Claude（或自己填入 `data/config.js` 的 `portalUrl`
   後 git push），官網就會出現「填報系統」按鈕。
